# SkyCast - Modern Weather Application

SkyCast is a professional, responsive, and production-ready weather application built with modern web technologies.

## Features
- **Real-time Weather Data**: Current temperature, feels-like, humidity, wind speed, and UV index.
- **7-Day Forecast**: Comprehensive daily forecast for the week ahead.
- **Dynamic Search**: Instantly find weather information for any city globally.
- **Geolocation Support**: Access local weather with a single click using browser geolocation.
- **Glassmorphism UI**: A premium, layered design featuring the default teal branding (`#00b7af`).
- **PWA Ready**: Offline support and installable on mobile devices via service worker and manifest.
- **Optimized Performance**: Integrated cache-busting and no-key API access via Open-Meteo.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Modern layouts using Flexbox/Grid and backdrop filters.
- **Vanilla JavaScript (ES6+)**: Asynchronous API handling and dynamic DOM updates.
- **Open-Meteo API**: High-accuracy weather data without the need for API keys.

## Getting Started
Simply open `index.html` in any modern web browser to start using SkyCast.
